# Prev22 Compiler
A compiler made for programming language prev22 as part of undergraduate course Compilers at [Faculty of Computer and Information science](https://www.fri.uni-lj.si/en) in Ljubljana!

# How to run
    $ cd prev22
    $ make clean && make

    $ cd prev22/prg
    $ make knapsack

    $ cd prev22/prg
    $ ./mmixal -x knapsack.mms
    $ ./mmix knapsack.mmo
